Installation:
npm install -g node-sass
npm install -g sass-convert   <only if you need conversion>

Commands:
node-sass --w scss/ --o  ./     (As per the demo project, this command will watch changes in scss files under scss folder and write it to css)
sass-convert style.css style.sass  (to convert css into sass)


Useful links:
https://sass-lang.com/
https://www.npmjs.com/package/node-sass
